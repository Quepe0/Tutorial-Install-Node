# Tutorial Install Node X Version

INSTALLATION:
- apt-get update
- apt-get upgrade
- apt-get install curl
- curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.2/install.sh | bash
- chmod +x ~/.nvm/nvm.sh
- source ~/.bashrc 
- nvm install 18
- node -v

[Yes now you have successfully installed node version 18]
